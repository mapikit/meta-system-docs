Guides

If you're on this page, you're much ready to start working on something incredible! Is it a brand new system? Or is it the killer Addon everybody was waiting for? Let's start talking about the possibilities.

Before we proceed, if you haven't done it yet, we strongly recommend to read the MSYS documentation, so you will truly grasp the fundamentals.

Disclaimer

The authors of the Addons are the responsible for documenting and maintaining what are its behaviors, and its API. If you're an author, There are people who can help you at our discord.

Guides to Creating Systems#

If you're already familiar with the configuration of Meta-System and just need some inspiration, we have some examples and in-depth explanations of configurations at the Tutorials tab. If you want to learn it, we also have a page to teach you How to Think With Meta-System

Guides to Writing Addons#

You just found yourself wanting your system to do more. The built in functions and what the community has built so far does not suit you well enough, or you thnk there is some secret logic you require and don't want it being passed around. If this describes your situation, you'll need to create your own Addon.

To do so, startby creating your new repository and be sure to install meta-system with npm i -g meta-system and then head to the Creating Addons page.

Guides to Contributing to Meta-System#

If would like to help, you are more than welcome to! There are a couple of ways in which you can help on making Meta-System more awesome, and they are all explained under this documentation page.
